## Prerequisite
 - Node.js
 - Python3

type following commands at root directory of this project. 
```shell script
npm install -g yarn
npm install
```

```shell script
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
```  
 
## To start server
 Move to root directory then 
 type ```yarn start-api```
 then
 type ``yarn start``
